Process

My process for this project was to get functionality working first.
I wanted to ensure that every feature was working, testing until it was right.
Once the functionality of the website was in place, I began to implement the css,
trying out different ways of displaying items and making adjustments until I was
at a place I liked.

Resources

For tutorials on javascript and css, I used W3C Schools and Stack Overflow.

The images I have saved manually come from the PokeAPI.

Special Features

The only effort I want to make aware is how the search term gets fixed.
The API has a specific way that it wants names to be written, otherwise it throws an error.
I added some code so that so long as the name is spelled right, the pokemon will be found.
